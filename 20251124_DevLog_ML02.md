## 4. 기술적 세부 사항

### 4.1. .gitkeep 파일 사용 이유
Git은 기본적으로 빈 디렉토리를 추적하지 않습니다. 따라서 빈 폴더 구조를 원격 저장소에 유지하려면 각 디렉토리에 더미 파일(.gitkeep)을 추가해야 합니다.

**App/.gitkeep 내용:**
```
# This file ensures the App directory is tracked by Git
# App directory will contain the built application files
```

**Release/.gitkeep 내용:**
```
# This file ensures the Release directory is tracked by Git
# Release directory will contain release-ready packaged files
```
